Scope

🚨 This is not a workshop on how to use GIS mapping software. Please consider referring to our other in-person learning opportunities or follow-along materials from previous events.

🚨 Today will focus on how to find relevant source data for a GIS project.

Goals

  1. Understand the value of a few common GIS data approaches.
  2. Get a feel for the flow and pace of working with GIS data.
  3. Leave knowing which resources you can rely on for your project.

Sample data

You can download and explore sample datasets related to this activity from the workshop data homepage, hosted on the Open Science Framework (OSF.io)

  1. Visit the workshop data homepage.
  1. Click the three vertical dots icon and select Download.
  1. The folder that downloads to your computer contains sample data from all the workshop activities. It is a zipped or compressed file. In order to use it, you will have to double-click it on Mac or right-click → Extract or Uncompress on a PC.

If you’d like to follow along in class using ArcGIS Pro, here are some tips for obtaining that software:

  • Harvard Center for Geospatial Analysis (CGA) manages ESRI licenses on the Harvard campus; check out CGA’s ArcGIS Pro installation instruction.
  • Harvard Map Collections library guide, ArcGIS Pro Access & Support at Harvard.

Outline
