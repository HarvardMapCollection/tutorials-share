Project datasets

About

Harvard University Planning and Design (HUPAD) has provided access to the following datasets, which you can use in your Re-Wilding projects:

  • Yard trees
  • Hardscape (includes surface materials and pathways)
  • Street blocks
  • Greenspace
  • Buildings

Preview

Download instructions

To download a copy of these datatsets to use in your project:

  1. Visit this shared class map
  2. In the map legend, click the ellipsis next to the layer you want to download
  3. Select Export and to GeoPackage
