Finding GIS data for your project [demo in ArcGIS Pro]

Hands-on class activity covering common ways to find, create, or augment geospatial data. This activity presupposes some knowledge about Geographic Information Systems (GIS) and spatial methods.

5 activities

Activities

  1. Welcome

    Homepage including slides, data, and workshop objectives.

  2. 1. Portals

    Learn how to find and interpret materials in open GIS data portals.

  3. 2. Coordinates

    Learn how to create coordinate data compatible with GIS software.

  4. 3. Georeferencing

    Understand the value of georeferenced maps and how to start using them in a GIS project.

  5. 4. OpenStreetMap

    Learn how to access crowdsourced GIS data for situations where data is difficult to find.
