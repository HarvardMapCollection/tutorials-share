If you try to drag and drop a `.geoJSON` or `.json` file into the ArcGIS map interface and receive an error, try these steps.

1. Navigate to the `Analysis` tab.
2. Select `Tools` or the red toolbox icon in the `Geoprocessessing` section of the `Analysis` ribbon.
3. Search for `JSON to Features`.
4. Specify your source, output, and geometry parameters.
5. Select `Run`.
